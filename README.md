# 🚀 Digital Employee AI

A modern AI-powered SaaS Content Generator built with React, TypeScript, and Groq API.

## ✨ Features

- 🤖 AI Content Generation
- 💼 LinkedIn Posts
- 📸 Instagram Captions
- 🐦 X (Twitter) Posts
- 📘 Facebook Posts
- 🧵 Threads
- 📰 Blog Posts
- 📧 Professional Emails
- 📨 Cold Emails
- 📝 Newsletters
- 📦 Product Descriptions
- 🎥 YouTube Descriptions
- 📄 Resume Summary
- 💼 Cover Letter Generator
- 📢 Press Releases
- 🎧 Customer Support Replies

## 🎨 Modern UI

- Dark / Light Mode
- Responsive Design
- Beautiful Dashboard
- Markdown Rendering
- Loading Animations
- Toast Notifications
- History Sidebar
- Advanced Settings
- Word Count
- Character Count
- Reading Time
- Export as TXT
- Export as Markdown
- Print to PDF
- Keyboard Shortcuts

## 🧠 AI Polish Actions

- Improve
- Rewrite
- Expand
- Shorten
- Professional Tone
- Friendly Tone
- Funny Tone
- Persuasive Tone
- Add Emojis
- Fix Grammar
- Generate Hashtags
- Generate CTA

## ⚙️ Tech Stack

## 🚀 Getting Started

```bash
git clone https://github.com/sinyaakumarii/digital-employee-ai.git

cd digital-employee-ai

npm install

npm run dev
```

## 🔑 Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key_here
```
